Community Database Management for data listing, export data, import data and google charts from database.

== Description ==

Community Database is for Adding and retreiving single data, csv data from database from backend and Google Charts showing.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/hct-community-database-management` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress activating and installing your plugin.
3. For adding multiple data use "hct_community_sample.csv" available in plugin folder
4. If your are exporting the csv file, please do not upload the same format again, use the recommended file in above point.
